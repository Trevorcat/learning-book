## 示例

### 功能名称
AI 对话面板

### 功能描述
在侧边栏创建一个聊天界面，用户可以输入问题，插件调用 Claude API 获取回答。

### 需求细节
- [ ] 创建 LearningView 类，继承 ItemView
- [ ] 实现聊天界面 UI（输入框 + 消息列表）
- [ ] 调用 Claude API 发送消息
- [ ] 本地存储对话历史
- [ ] 支持 Markdown 渲染

### 相关文件
- 需要修改：main.ts（注册视图）
- 需要创建：src/views/LearningView.ts

### 参考
- Obsidian View API：https://docs.obsidian.md/Plugins/User%20interface/Views

### 验收标准
1. 侧边栏显示聊天界面
2. 能发送消息并收到回复
3. 对话历史保存到本地
4. 支持 Markdown 格式显示
