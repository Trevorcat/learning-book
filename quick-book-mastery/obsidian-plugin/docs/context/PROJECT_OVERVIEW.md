# Quick Book Mastery - Obsidian Plugin

## 项目背景

这是 [learning-book](https://github.com/Trevorcat/learning-book) Skill 的 Obsidian 插件实现版本。

原 Skill 是一个**对话式学习引导系统**，通过 AI 与用户的对话帮助用户快速掌握一本书的核心知识（30分钟-2小时）。

## 核心概念

### 两种学习模式
1. **快速模式（Fast Mode）**
   - 时长：30-60分钟
   - 适合：快速了解全书框架、已有基础
   - 方式：AI 先介绍核心，边学边提问，快速推进

2. **伴读模式（Companion Mode）**
   - 时长：细粒度，按小节推进
   - 适合：系统深入学习、知识点密集的书籍
   - 方式：AI 像老师一样带领，先讲解后提问，主动识别盲区

### 四阶段学习流程
```
┌─────────┬─────────┬─────────┬─────────┐
│  阶段1   │  阶段2   │  阶段3   │  阶段4   │
│  准备   │  概览   │  深入   │  巩固   │
│ 5-10min │ 10-15min│ 15-30min│  5-10min │
├─────────┼─────────┼─────────┼─────────┤
│明确目的 │检视阅读 │费曼复述 │间隔重复 │
│激活先验 │二八法则 │主动回忆 │自测检验 │
│设定目标 │框架构建 │苏格拉底 │行动计划 │
└─────────┴─────────┴─────────┴─────────┘
```

### 学习方法论
- **费曼学习法**：用自己的话解释概念，识别知识盲区
- **主动回忆**：通过提问检验记忆，而非被动重读
- **间隔重复**：1天→3天→7天→21-30天的复习计划
- **苏格拉底式提问**：澄清、假设质疑、证据追问、视角转换

## 技术栈

- **语言**: TypeScript
- **构建**: esbuild
- **平台**: Obsidian API
- **可选集成**:
  - Excalidraw 插件（手绘框线图）
  - Spaced Repetition 插件（间隔重复）
  - Dataview（统计面板）

## 目标用户

- 需要快速了解一本新书的读者
- 希望系统性学习某本书的学习者
- 准备读书分享或书评的人
- 需要建立某领域知识框架的初学者
