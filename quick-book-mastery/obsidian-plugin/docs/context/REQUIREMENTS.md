# 需求文档 (Requirements)

## 功能需求

### MVP (最小可行产品)

#### 1. 书籍管理
- [ ] 创建书籍学习项目（文件夹+模板）
- [ ] 列出所有正在学习的书籍
- [ ] 显示每本书的学习进度（四阶段进度条）

#### 2. 学习模式选择
- [ ] 开始新书籍时选择：快速模式 / 伴读模式
- [ ] 模式说明和推荐

#### 3. 四阶段学习流程
- [ ] 阶段1：准备与目标设定
  - 书名、作者、目标、时间、先验知识输入表单
  - 生成学习档案

- [ ] 阶段2：概览与框架构建
  - 检视阅读引导（目录→序言→章节首尾）
  - 框架图生成（Markdown 大纲/表格）
  - 关键20%内容标记

- [ ] 阶段3：深入与理解
  - 核心概念列表
  - 费曼复述记录区
  - 知识盲区标记
  - 主动回忆问题列表

- [ ] 阶段4：巩固与应用
  - 间隔重复计划表
  - 自测清单
  - 行动计划（SMART目标）

#### 4. AI 对话面板
- [ ] 侧边栏聊天界面
- [ ] 支持用户输入 API Key（本地存储）
- [ ] 调用 Claude API 进行对话
- [ ] 保留对话历史

### 进阶功能 (V2)

#### 5. 可视化集成
- [ ] 集成 Excalidraw 创建框架图
- [ ] 思维导图导出

#### 6. 间隔重复集成
- [ ] 与 Spaced Repetition 插件联动
- [ ] 自动生成复习卡片

#### 7. 统计面板
- [ ] 学习时长统计
- [ ] 完成书籍统计
- [ ] 知识盲区热图

#### 8. PDF 集成
- [ ] 分屏阅读（左边PDF，右边笔记）
- [ ] 高亮同步（选中文本→AI解释）

## 非功能需求

### 性能
- 启动时间 < 2秒
- 切换书籍 < 500ms
- AI 对话响应 < 3秒（取决于API）

### 兼容性
- 支持 Obsidian 0.15.0+
- 桌面端（Windows/macOS/Linux）
- 移动端（iOS/Android）基础功能

### 数据存储
- 所有数据本地存储（Markdown 文件）
- 不使用远程数据库
- 支持 Obsidian 原生同步

### 隐私
- API Key 本地加密存储
- 书籍内容不上传
- 可选择本地模型（未来）

## 技术要求

### 开发规范
- TypeScript 严格模式
- ESLint + Prettier
- 单元测试（Jest）
- 文档注释

### 依赖限制
- 核心：obsidian API
- UI：原生 Obsidian 组件（无需 React/Vue）
- HTTP：标准 fetch 或 electron 的 net
- 存储：Obsidian Vault API
